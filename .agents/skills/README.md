# Skills Directory

This directory contains reusable skill files for the AI agent. Each subdirectory has a `SKILL.md` file that defines the skill's behavior and references.

## Available Skills

| Skill Name | Description |
|---|---|
| **angular-developer** | Generates Angular code and provides architectural guidance. Covers components, signals/forms, DI, routing, SSR, accessibility, animations, styling, testing, and CLI tooling. |
| **angular-new-app** | Creates a new Angular app using the Angular CLI. Includes guidelines for scaffolding, configuration flags, and best practices. |

## How to Use

To load a skill, use `read_skill("<skill-name>")` or read the `SKILL.md` file directly from its subdirectory.
